This repo has examples of how to use firebase. To learn more about any of these actions you can read the Firebase docs.

Before you can use this repo you need to set it up with your own Firebase. Put your own firebase config in to the fb_setup.js script and enable it as an "Authorised domain" in Firebase. The fb_initialise function is called when the page loads (by script.js)

Files

index.html

  • The landing page
  • Where you will put buttons
  • Where you will trigger database actions
  • where you will get responses from the database

fb_setup.js

  • The script that sets up the database
  • This is where you will paste the configuration data from firebase
  • You won't touch this file much

fb_io.js

  • This is where all your database functions will go
  • You will add to this as you learn more skills
  • This is your own database accessing library

script.js

  • This is the script that is linked to the buttons on the page
  • This page calls functions in fb_io.js
